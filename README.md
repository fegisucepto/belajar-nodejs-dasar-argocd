# Belajar Node.js Dasar

Project Dasar Node.js + Express.

## Install
```bash
npm install
```

## Menjalankan
```bash
npm run dev
# atau
npm start
```

## Endpoints
- GET  /api/users
- GET  /api/users/:id
- POST /api/users { name, email }
- PUT  /api/users/:id
- DELETE /api/users/:id
- GET  /health
